# Company Brain release checklist

- [ ] TypeScript passes.
- [ ] Unit and integration tests pass.
- [ ] Production build passes.
- [ ] Database migrations are applied and advisor checks are clean.
- [ ] Tenant-isolation regression tests pass.
- [ ] Recommendation evidence and approval gates are verified.
- [ ] Simulation writes cannot mutate operational tables.
- [ ] Plan generation is separate from activation.
- [ ] Skills cannot execute without declared permissions.
- [ ] External execution still requires idempotency and receipts.
- [ ] Company Brain page works on desktop and iPhone WebKit.
- [ ] No demo or synthetic metric is presented as production data.
