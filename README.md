# candy-agents-v3

نسخة جاهزة للنشر على Vercel من لوحة “النجمة الذهبية — نظام الـ Agents”.

## ملاحظة أمان مهمة
تم تعطيل مفاتيح Google Sheets ونداءات الذكاء الاصطناعي المباشرة من المتصفح لأن وضعها داخل الواجهة يكشفها للجميع.
النسخة الحالية تعمل بواجهة كاملة وتحفظ السجلات محلياً داخل المتصفح.

## النشر على Vercel
1. ارفع هذا المجلد إلى GitHub.
2. افتح Vercel واختر Import Project.
3. اختر الريبو.
4. Framework: Vite.
5. Build command: npm run build.
6. Output directory: dist.
7. Deploy.
