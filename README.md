# Golden Star Enterprise OS

Next.js internal company management platform.

## Deployment status

Clean redeploy trigger for Vercel from main branch.

## Features

- Apple style executive dashboard
- Employees management
- Tasks board
- Daily logs
- Approvals center
- Notifications
- Activity tracking
- Supabase database layer
- Google Sheets export service
- Health check endpoint

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Health check

```txt
/api/health
```
